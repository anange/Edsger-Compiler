# Edsger

## How To Compile it

	cd src
	make 

## How To Test it

	cd src
	make test1
	make test2

## Tools
* opam (package manager)
* utop (opam install utop)
* core (opam)
* Menhir (opam install menhir) 
* Ocamllex
